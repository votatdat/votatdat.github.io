Danh sách đầy đủ bài học ở đây.

Thông thường khi tạo một Django project thì database mặc định là SQLite, nhưng thực tế để lên production hiếm khi nào người ta sử dụng SQLite, người ta hay sử dụng PostgreSQL.
(Theo mình quan sát thấy thì Python dev thích PostgreSQL, Java dev thì thích MySQL)
Tutorial này sẽ tạo Docker có chứa PostgreSQL, giới thiệu để đi tiếp cho các tutorial sau.

Cài đặt

{% highlight python %} $ mkdir postgresql && cd postgresql # Tạo folder làm việc và vào folder $ pipenv install django==2.2.7 # Cài đặt django bằng pipenv $ pipenv shell # Vào môi trường (postgresql) $ django-admin startproject postgresql_project . # Tạo project, lưu ý dấu . ở cuối câu lệnh (postgresql) $ python manage.py migrate # migrate (postgresql) $ python manage.py runserver # runserver {% endhighlight %} Sau đó, bật trình duyệt vào trang http://127.0.0.1:8000/ thấy dòng "The install worked successfully! Congratulations!" là OK.
Chúng ta ấn Ctrl + C để để dừng, và dùng lệnh ls để xem các file và directory. {% highlight python %} (postresql) $ ls Pipfile Pipfile.lock db.sqlite3 manage.py postgresql_proje {% endhighlight %}
